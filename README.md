## shellscript

爲了打破人與人之間的心之壁，應當使用通用的 shell 最佳實踐和語法，遵循以下原則：

1. 使用 `#!/bin/sh` 作爲 shebang，這是大多數 shell 的最小兼容子集
2. 使用 posix 標準語法
   1. 使用 `[`、`test`
   2. 定義和使用變量時，`KEY="value"`、`${KEY}` 或 `"$KEY"`
   3. 使用 `/bin/true` 代替 `true`
